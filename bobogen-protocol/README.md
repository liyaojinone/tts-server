# bobogen-protocol

Shared protocol models for BoboGen model services.

## Purpose

This package defines the common HTTP contract shared by model-specific generation services:

- voice listing
- legacy synthesize request/response schema
- unified generate request/response schema
- common parameter fields
- error response shape

## Install

Install into each model service environment:

```bash
pip install -e E:\AiModel\bobogen-server\bobogen-protocol
```
